02 Introduction to Web Design with Bootstrap

This is to showcase how to build a grid, using Rows and Columns with Bootstrap.

A page has a total of 12 divisions in it. We can section off the grid using [class="col-md-X"] where X is how big we want each column to be.

"col-md-6" this cell takes up half the page width
"col-md-2"
"col-md-4" or 1/3rd the page
"col-md-12 block" = this takes up the whole length of the page & Block is a Class identifed with CSS
